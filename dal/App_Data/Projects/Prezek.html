Implementation & Tecnologies

Crawling: Data was extarcted from the web by means of crawler written in nodejs. Data was read in raw html format, then parsed with nodejs jquery module, after which yandex geocode api was used to link addressed with their geo coordnates.

Single page application: Client apllication build upon knockoutjs framework using coffee-script as a single web page application.

Caching: To helps system serve user data fast and relaiable memcached library is used.
