Summary & Objectives

Data-Avail WinForms is a MV* framework on the base of NET. Windows Forms libraries. Default NET. Windows Forms application provides for developer event driven code paradigm with design time utilities and code behind model, Data-Avail WinForms framework shift this paradigm to MV* code implementation.

The framework also makes it possible to implement whole application using only XML templates which describes data layer, user interfaces and security settings.

Implementation

The library is implemented with C#. User interface components use DevExpress WinForms Library. Access to data layers implemented with DevArt data providers.

The library implemented as Model View Controller framework which provides following general interfaces:

  1. Controller
  2. ICommand
  3. IDataAdapter
  4. Binding

Features

  1. MV* style framework for data centric Windows Forms applications
  2. Build applications with no coding, using only XML templates
  3. Plugin system for UI components
  4. Moduled code implementation
  5. Use modern 3rd party components and libraries
  6. Open source
