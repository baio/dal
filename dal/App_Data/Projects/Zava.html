Implementation

  • ASP.MVC 4.5 web application
  • Main database - MongoDB
  • Account database - MS SQL
  • Static files stored on Amazon S3

Features

  • News list (add, edit, comment).
  • Automatically retrieve news from google feed by keyword, then admin could confirm them. This is separated application written with nodejs and hosted on heroku.
  • Faq list (add, edit, comment).
  • Authorization - Admin role allow moderate, accept and confirm users news and questions
  • Mail list - latest new are sent via MailChimp
